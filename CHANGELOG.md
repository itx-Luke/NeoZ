## [0.1.0] - 2025-12-31

### Added
- **DRCS (Dynamic Resolution Calibration System)** — Automatic sensitivity adjustment
- **AI Advisor** — Gemini-powered sensitivity recommendations
- **ADB Connector** — Emulator detection and connection management
- **Script Runner** — Execute shell scripts on connected devices
- **Modern QML UI** — Dashboard, settings, history, and advanced AI views
- **Theme Support** — Multiple themes with custom wallpaper support
- **Crosshair Detection** — Aim assist state monitoring
- **Smart Pointer Refactoring** — Improved memory management

### Infrastructure
- CMake build system with Qt 6.6+ support
- Unit tests for DRCS, Logger, Sensitivity, and NeoController
- GitHub Actions workflow for Windows builds
